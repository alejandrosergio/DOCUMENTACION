ATTR()

Teoria

attr() se suele combinar con la función each() a la hora de recorrer elementos y trabajando con selectores. Otras funciones similares son estas:
css()
animate()
addclass() y removeclass()
val()

Cuando attr() se usa para obtener el valor de un atributo devuelve el valor del primer elemento encontrado
